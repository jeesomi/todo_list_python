# todo_list_python
Python To-Do List Manager (OOP & CSV)

A robust, object-oriented Python application designed to manage daily tasks efficiently. This project demonstrates core programming principles including Object-Oriented Programming (OOP), File I/O, and Data Persistence using CSV files.
